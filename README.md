This project uses deep learning to predict how atmospheric conditions evolve over time based on global geopotential height data at the 50 hPa and 500 hPa pressure levels. I implemented both a baseline CNN and a ResNet architecture, along with physics-inspired approaches using Euler and RK4 integration for multi-step forecasting. The data was preprocessed with normalization and split into training, validation, and test sets, and models were trained using mean squared error loss in PyTorch. I evaluated performance on both single-step and multi-step predictions, finding that while standard neural networks performed well short-term, physics-informed models provided better stability over longer forecasting horizons.
